# RestTemplate

- spring 3.0 부터 지원한다.
- 스프링에서 제공하는 http 통신에 유용하게 쓸 수 있는 템플릿
- HTTP 서버와의 통신을 단순화하고 RESTful 원칙을 지킨다.

<br/>

- RestTemplate
  - Spring 3부터 지원, REST API 호출이후 응답을 받을 때까지 기다리는 동기 방식.
- AsyncRestTemplate
  - Spring 4에 추가된 비동기 RestTemplate이다.
- WebClient
  - Spring 5에 추가된 논블럭, 리엑티브 웹 클라이언트로 동기, 비동기 방식을 지원한다.
